# Md Abdul Basit - Personal Portfolio

A sleek, modern, FAANG-level responsive personal portfolio showcasing projects, experience, education, and skills. Built with **React** and **Tailwind CSS**.

## ✨ Features
- **Dark Modern Theme**: High-contrast, glowing UI designed for aesthetics and readability.
- **Responsive**: Fully optimized for mobile, tablet, and desktop views.
- **Glassmorphic UI**: Beautiful blurred sidebars, glowing backgrounds, and layered cards.
- **Smooth Animations**: Interactivity powered by `framer-motion` for a premium feel.
- **Performance Optimized**: Built using Vite for instant server start and lightning-fast HMR.

## 🚀 Getting Started

### Prerequisites
Make sure you have Node.js and NPM installed.

### Installation & Setup
1. **Navigate to the directory**:
   ```bash
   cd portfolio
   ```

2. **Install dependencies**:
   *(Already done for you: Tailwind, Framer Motion, and Lucide React are preinstalled)*
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **View in Browser**: 
   Open `http://localhost:5173/` in your browser.

## 🛠️ Tech Stack
- **Framework**: React.js, Vite
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React
- **Animations**: Framer Motion

## 📁 Project Structure
- `src/components/*` - Contains all modular UI sections (Hero, About, Projects, Gaming, etc.)
- `src/App.jsx` - Aggregates the layouts.
- `src/index.css` - Custom Tailwind and CSS configurations.
